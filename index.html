What is Node Todo?

Node Todo will guide you through your first contribution to Node.js core. As long as you've worked a little bit with Node.js, you can do this!

Node Todo is a fully volunteer effort independent of the Node.js project.

This website is open source and you can fork it on GitHub. The website is based on the DogpatchJS website code.
